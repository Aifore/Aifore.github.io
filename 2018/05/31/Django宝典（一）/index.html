1. 简介：

Flask、Django、Tornado、Pyramid、Bottle、Web2py、web.py

MVC：模型-视图-控制器

MTV：

Model：可以通过ORM(对象关系映射)方便地对各类数据库进行操作，而不用针对不同数据库写不同的Sql语句，是面向对象的编程；

Template：数据展示层；

View：处理层；

目标：通过控制器，将数据和显示分离——[模型（数据）和视图（显示）解耦合]

同一个视图可以加载不同的模型，同一个模型也可以显示成不同的视图。稍具规模的系统都会使用MVC架构或者它的变体（MVP、MVVM等），是对面向对象设计原则中迪米特法则的一个最好的践行。

2. 配置步骤

2.1 配置项目环境

mkdir hello_django 创建项目目录

cd hello_django/ 切换到项目目录

  • python -m venv hd_venv 使用venv模块创建虚拟环境，目录名hd_venv

cd hd_venv/Scripts/

  • activate.bat 激活虚拟环境

    source hd_venv/bin/activate 激活虚拟环境（linux）

    pip install django 使用pip安装django

    django-admin –version 通过安装django时安装的脚本工具django-admin检查django版本

  • django-admin startproject hello_django . 创建项目

    python manage.py startapp hrs 创建服务

  • python manage.py runserver 0.0.0.0:80 运行服务-自动匹配本机地址
