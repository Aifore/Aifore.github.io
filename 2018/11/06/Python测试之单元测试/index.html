4. Django运行单元测试

  1. 测试项目中所有的应用

python manage.py test

  1. 测试项目中单独的应用

python manage.py test app01

  1. 运行项目中某个应用的测试文件中的一个Case

python manage.py test app01.test2.AuthorTestCase

  1. 运行项目中某个应用的测试文件中的一个Case中的其中一个测试方法

python manage.py test app01.test2.AuthorTestCase.test_insert_data

  1. 运行单元测试结束时不自动删除测试数据库（保留测试数据库）

python manage.py test app01 –keepdb

5. API权限问题的解决

在测试API的时候，往往需要进行用户登录才有权限调用，此时需要指定登录用户来解决接口调用的权限问题

6. Celery异步任务的测试

在代码中几乎肯定是会有celery异步任务，若想对异步任务进行单元测试。可以将CELERY_ALWAYS_EAGER=True, BROKER_BACKEND=’memory’
