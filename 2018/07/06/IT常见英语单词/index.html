Veröffentlicht am 2018-07-06 | Edited on 2018-05-26 | in 其他笔记

IT常见英语单词

specify：vt. 指定；详细说明；列举；把…列入说明书

track：vt. 追踪；n. 轨道

record：vt. 记录，记载；标明；将…录音

discard：vt. 抛弃；放弃；丢弃

caution：n. 小心，谨慎; 警告，警示; vt. 警告

recursive：adj. [数] 递归的；循环的

concatenate：vt. 连结；使连锁；adj. 连结的；连锁的

terminate：vt. 使终止；使结束；解雇 vi. 结束，终止；结果

promote：vt. 促进；提升；推销；发扬 vi. 成为王后或其他大于卒的子，上位

generate：vt. 使形成；发生；生殖；产生物理反应

deploy：vt. 配置；展开；使疏开 vi. 部署；展开 n. 部署

layout：n. 布局；设计；安排；陈列

MySQL

SQL(Structured Query Language): 结构化查询语言

DDL(data Definition Language): 数据定义语言

create(创建) / drop(删除) / alter(修改)

DML(Data Manipulation Language): 数据操作语言

insert(插入) / delete(删除) / update(更新)

DQL(Data Query Language): 数据查询语言

select(选择)

DCL(Data Control Language): 数据控制语言

grant(授予权限) / revoke(召回权限) / begin(开启事务) / commit(提交事务) / rollback(回滚事务)
