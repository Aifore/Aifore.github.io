所以可以给出大概的结论：

  1. 因为装饰器是从内到外包装的，装饰器C在最内层，最先包装，所以加载模块时执行的顺序是由下往上，即装饰器的包装过程是C 👉 B 👉 A；

  2. 因为函数却是从外到内执行的，就像一层一层地拆包装一样，最外的一层包装（装饰器A）最先执行，所以执行被包装的函数时的顺序是由上往下，即装饰器代码的执行过程是A 👉 B 👉 C；

    • A(B(C(pd)))从外到内一层层执行，依次调用a_wrap，b_wrap，c_wrap最终返回被装饰的pd函数对象。
