1. datetime模块

  1. 其实可以不用将时间格式化成字符串，一是不管时间被转换成什么格式的字符串，最后django在页面解析的时候都会解析成2018年7月7日 00:03这种格式；二是django可以直接解析[datetime.datetime]时间对象。

    • 在mysql中执行 SET GLOBAL time_zone=’+08:00’；

    • django默认是有运用时区的，需要在设置中把USE_TZ=False；

Django宝典（一）

发表于 2018-05-31 | 更新于 2019-04-05 | 分类于 Django |

1. 简介：

Flask、Django、Tornado、Pyramid、Bottle、Web2py、web.py

MVC：模型-视图-控制器

MTV：

Model：可以通过ORM(对象关系映射)方便地对各类数据库进行操作，而不用针对不同数据库写不同的Sql语句，是面向对象的编程；

Template：数据展示层；

View：处理层；

目标：通过控制器，将数据和显示分离——[模型（数据）和视图（显示）解耦合]

同一个视图可以加载不同的模型，同一个模型也可以显示成不同的视图。稍具规模的系统都会使用MVC架构或者它的变体（MVP、MVVM等），是对面向对象设计原则中迪米特法则的一个最好的践行。
